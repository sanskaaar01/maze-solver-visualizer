# Maze Solver Visualizer (BFS vs Dijkstra)

## 📌 Overview
This project is an interactive maze solver built using Python and Tkinter.

It demonstrates how BFS and Dijkstra algorithms behave on a maze modeled as a graph.

---

## 🚀 Features
- Interactive GUI (click to set start and end)
- BFS (shortest path in steps)
- Dijkstra (shortest path in cost)
- Weighted maze visualization
- Path animation
- Algorithm comparison (steps, cost, time)

---

## 🧠 Concepts Used
- Graph representation
- Breadth-First Search (BFS)
- Dijkstra’s Algorithm
- Priority Queue (Heap)
- Algorithm analysis

---

## 📊 Comparison Logic
- BFS → shortest path (ignores cost)
- Dijkstra → minimum cost path
